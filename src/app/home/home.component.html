Curtis achieved dean's honour every Camosun academic term.

Curtis was part of a student team working with Cloud Compass Computing Inc. The project is StackButton and it provides a status-board for monitoring multiple cloud-deployed software tools. Curtis learned documentation, reporting, project management strategies, system troubleshooting, process development, and system analyst workflows.
