Technology

Curtis graduated from Camosun College's Computer Systems Technology in the summer of 2017 with honours and is working towards building a long-term software development career.

Working at Schneider Electric on a manufacturing team, Curtis is building a tool for managing manufacturing change process using the MEAN stack. In manufacturing, one flashing light could communicate many things so there are many fascinating user communication challenges available. Many concepts like poka-yoke are essential, so understanding how to create mistake proof systems is a must. Curtis is excited to be jumping into his career in this company.

In 2016 Curtis contributed on a student team working with Cloud Compass Computing Inc. The project is StackButton and it provides a status-board for monitoring multiple cloud-deployed software tools. StackButton is written in Angular 4.0 and is deployed in a Docker container. It is designed for Red Hat's OpenShift platform and collects API information.

At the BC Public Service Agency, Curtis joined the Time and Leave Project team for a six month co-op as application support during the product rollout. Curtis learned documentation, reporting, project management strategies, system troubleshooting, process development, and system analyst workflows. The core focus of the work was supporting the project. Curtis was responsible for troubleshooting access issues post-deployment within the BC Government internal network. Curtis’ pet project was the mobile enabled version of BCGov’s MyHR built on PeopleMobile built by Greyheller; this involved working with multiple vendors and contractors to create a user-friendly mobile version of the Time and Leave System.
