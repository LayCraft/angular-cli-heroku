About this site:

This site serves two purposes: a place for a prospective employer to get information, and a place where Curtis LayCraft can learn/experiment with new technologies. It is built using feature branching on Github.

This is a node.js web application built in Angular. The UI is built with Bootstrap 4. It is being versioned through Github, and automatically deployed through a Heroku pipeline.
